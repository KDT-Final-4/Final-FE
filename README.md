# Final-FE
Final 프로젝트의 프론트엔드(React 18 + TypeScript + Vite + Tailwind CSS 4) 레포지토리입니다. 대시보드, 트렌드, 구성, 로그, 리포트, 스케줄, 사용자 관리 등 라우트 페이지를 제공하며 공통 UI 컴포넌트를 기반으로 구성됩니다.

## 프로젝트 실행 방법
1. `.env.example`을 참고해 프로젝트 루트에 `.env` 파일을 직접 작성합니다.
2. 의존성을 설치합니다.
   - `npm install`
3. 개발 서버를 실행합니다.
   - `npm run dev`
